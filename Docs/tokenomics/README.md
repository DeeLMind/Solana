# Solana 代币经济模型

## 1. SOL 是什么

`SOL` 是 Solana 网络的原生代币。

它主要承担几种角色：

- 支付交易手续费
- 参与 staking
- 为网络安全提供经济权重
- 作为生态内最基础的计价资产之一

## 2. 手续费结构

Solana 的手续费通常不像很多 EVM 链那样只靠统一 gas 竞价直观表达，但本质上仍然是在给网络资源定价。

常见你会接触到：

- base fee
- priority fee
- compute budget

### Base Fee

基础费用对应交易进入系统和消耗基础资源的成本。

### Priority Fee

优先费用于在网络拥堵时提高交易被优先处理的概率。

在高峰期，priority fee 的重要性会明显提升。

## 3. Compute Budget 与费用

交易执行不是无限制的。

Solana 用 compute units 去衡量资源消耗，因此：

- 更复杂的交易通常需要更高 compute budget
- 更高优先级通常意味着更高费用竞争

这让费用模型和运行时资源模型强相关。

## 4. 通胀与奖励

Solana 有原生通胀和 staking reward 机制。

直觉上可以理解成：

- 网络会发行一定新增代币
- 其中一部分通过 staking reward 分配给参与安全维护的 stake

因此 staking 不是单纯“锁仓”，而是和网络安全激励绑定在一起。

## 5. Staking Reward

质押奖励通常与这些因素相关：

- 委托 stake 数量
- 验证者表现
- commission
- 网络整体奖励参数

这也是为什么委托人选择验证者是一个经济决策，而不是只看名气。

## 6. 手续费销毁与经济平衡

从宏观上看，网络会在：

- 新增发行
- 手续费支付
- 部分费用销毁或再分配

之间形成一种经济平衡。

学习这一层时，最重要的是理解：

- 网络安全不是免费的
- 吞吐、费用、激励和节点运营成本必须一起看

## 7. MEV 语境

在 Solana 生态里，`MEV` 也同样存在。

它会影响：

- 交易排序
- 套利行为
- 抢跑和夹子问题
- leader 与交易流之间的经济关系

如果你后面研究 DeFi、基础设施或搜索者生态，MEV 是绕不开的话题。

## 8. 为什么代币经济学不是只给投资者看的

对开发者来说，代币经济模型直接影响：

- 用户交易成本
- 高频交互产品可行性
- staking 相关产品设计
- 协议补贴与商业模式

对审计人员来说，它还影响：

- 激励是否可被操纵
- 奖励分配是否可被套利
- 费用假设是否会在极端条件下失效

## 9. 一句话记忆

Solana 的代币经济可以简化理解为：

- **SOL 既是手续费资产，也是 staking 权重资产，同时承担网络安全激励的经济基础。**
