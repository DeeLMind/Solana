# Solana 上的代币总览

来源：

- https://solana.com/zh/docs/tokens

## 代币在 Solana 上怎么表示

Solana 上的代币通常指 SPL Token。  
理解代币体系时，最重要的是区分这几个对象：

- `Token Program`
- `Mint Account`
- `Token Account`
- `Associated Token Account`

## 一句话理解

- `Mint Account`：定义“这是什么币”
- `Token Account`：记录“谁持有多少这个币”
- `Associated Token Account`：某钱包持有某个 mint 的默认 token account

## 两个主要 Token Program

官方文档提到有两个主要程序：

- 原始 `Token Program`
- `Token Extension Program (Token-2022)`

可以粗略理解为：

- 原版：基础、稳定、兼容广
- Token-2022：在保留基础能力上增加扩展特性

## 一个最小例子

假设你发行一个积分币 `POINT`：

1. 创建一个 `Mint Account`
2. 给 Alice 创建一个 token account
3. 铸造 1000 POINT 到 Alice 的 token account

这时：

- `Mint Account` 记录总发行量
- Alice 的 token account 记录她持有多少

## 新手最容易混淆的点

- `钱包地址` 不等于 `token account`
- 同一个钱包可以拥有多个 token account
- 一个 token account 只对应一个 mint
